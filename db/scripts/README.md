# Scripts para el llenado de la base de datos 

Estos son los Scripts de python para llenar de datos **falsos** la base de datos y backend puede hacer sus pruebas!


crea un etorno virtual para correrlo y poder llenar tu base de datos
en el archivo config es donde podras definir las credenciales de tu base de datos local 

---

# Requisitos

- Python 3.9+
- MariaDB instalado localmente con la base de datos creada
- faker (libreria)
- pip
